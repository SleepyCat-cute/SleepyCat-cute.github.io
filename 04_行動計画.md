## 0. この計画の考え方

**原則1: まず自分を守る。** あなたは告発者でも監査官でもなく一人の看護師です。証拠集めより自衛が先です。
**原則2: 敵を作らず「管理者の負担軽減」として売る。** 最初に組む相手は疲弊した管理者。持ち込む話は「セキュリティ強化」ではなく「業務改善」の顔をさせる。
**原則3: すべての節目で記録を残す。** 口約束は消えます。日時・発言者・内容をメモし、重要な面談は録音し、合意事項は当日中にメールで文書化する。

**録音について**: 自分が参加する会話の録音は、相手の同意がなくても一般的に労働紛争等の証拠として用いられています。ただし個別状況で評価は変わり得るため、迷う使い方(第三者提供など)は弁護士に相談してから行ってください。

### 全体フロー
```
Phase0 自衛と現状把握(〜2週間) → Phase1 味方づくり(2〜4週間) → Phase2 正式提案(1〜2ヶ月)
                                                                        │
                                                    ┌───────合意 / 拒否・放置───────┐
                                                    ▼                              ▼
                                              導入・定着                    Phase3 エスカレーション
                                                                          (内部→外部→公益通報)
【全Phase共通】報復の兆候 or 心身への影響 → 撤退ラインで転職準備に切替
```
Phase 0の自衛行動(LINE利用停止・事実ログ)は一度きりでなく、以降も並行して続けてください。

## 1. Phase 0: 自衛と現状把握(今日から2週間)

- **今日から、自分は患者情報を個人LINEで送らない。** 受信済み履歴は慌てて削除しない(削除自体が証拠隠滅と誤解される種になる)。「自分から送らない」だけを徹底する。
- **私物ノートに事実ログを開始**: 日時・誰が・何があった・自分の対応、の4点。患者はイニシャル化し、患部写真等の現物は転載・保存しない。証拠収集で自分が漏えい者・不正な第三者提供者にならないための鉄則(01参照: 私物端末での患者情報の保管は、安全管理措置・従業者監督の観点で問題視される行為そのもの)。
- **自分の直近訪問の「実滞在時間」vs「記録・請求区分」の突合を開始**(手順は02の第4章参照)。目的は告発準備ではなく、自分自身が虚偽記載に巻き込まれていないかの確認。他人の請求内容の詮索は不要。
- **書類開示を求める**: 労働条件通知書・就業規則・個人情報保護規程・運営規程。「入職時の書類を整理したいので」と自然に切り出す。労働条件明示は労基法15条で人数に関係なく全員に義務、未交付はそれ自体が独立した違法(30万円以下の罰金、120条1号)。一方、就業規則の作成・届出義務(89条)は常時10人以上が対象のため、7名規模のこの職場には原則ない。「就業規則がない」ことと「労働条件通知書がない」ことは法的な重みが違う点を理解しておく(条文の詳細は `_research/A9_不正類型と労基_公益通報.md` 参照)。
- **確認チェックリスト**(同僚から自然に):
  - [ ] 運営母体・上部組織の有無
  - [ ] 苦情・通報窓口の有無
  - [ ] 管理者・事務局長の上司は誰か
  - [ ] 事務局長の労基署案件のその後
- **録音の準備**: アプリの動作確認を済ませ、「有事の面談は必ず録音する」と自分の中でルール化する。

## 2. Phase 1: 味方づくり(2〜4週間)

- **管理者を最初の味方に。** 提案の角度は「電子カルテ+情報共有ツールはオンコール一元管理・訪問割当・時間管理を自動化し、管理者自身が最も楽になる」。疲弊した管理者にとって「辞めるまでの残り期間を楽にする話」として設計する(実際にLINE WORKS等の訪問看護導入事例で管理負担軽減が報告されている。詳細03)。
- **パート職員の温度感を確認**: 個人LINEでのやりとりに不安を感じている人は必ずいる(日本看護協会もSNS利用のリスクを注意喚起)。雑談レベルで探りを入れ、イニシャルでログに残す。この段階で勧誘はしない。
- **言葉選びを徹底**: 「セキュリティ強化」ではなく「業務改善・請求ミス防止」。「不正をなくそう」ではなく「時間管理を分かりやすくしよう」。犯人探しに聞こえた瞬間、事務局長は防御姿勢に入り管理者も手を引く。

---

## 3. Phase 2: 正式提案(1〜2ヶ月)

- 説得資料(05)を使い、可能なら**管理者と連名**で事務局長に提案。「若手看護師の思いつき」ではなく「現場を預かる管理者も同意している話」にする。
- 入口は必ず**0円案**(既存MCSの個別アカウント化+運用ルール+誓約書、詳細03のプラン0)。「新しい物を導入してほしい」ではなく「連携クリニックからも頼まれている個別アカウント化を、社内的にも仕上げるだけ」という言い方にする。金銭に厳しい事務局長には、初期・月額費用ゼロであることが提案を通す最大の鍵。
- **0円案にはもう1点、時間区分確認の役割分担も一緒に盛り込む。** 訪問時間が計画から大きくずれた場合に、記録・請求区分の確認と修正を「誰が(事務か、訪問した看護師本人か)」行うかを明文化する提案(詳細は02の3章・6-1章参照)。ルールが存在しないこと自体が管理不備であり、費用がかからず角も立ちにくい提案として一緒に通しやすい。
- **面談は録音**。結果は当日中にメールで議事メモを送付: 「本日は〜という提案をし、〜という結論になったと理解しています。相違があればご指摘ください。」――これが最大の証拠化手段。
- **決裂パターン別の次の一手**:
  - 「検討する」で放置 → 2週間後を目安に再度メールで確認し記録を積む
  - 費用を理由に拒否 → 0円案である旨を再確認、それでも拒否ならその理由も記録
  - 一蹴される → その場で反論せず、感情的対立を避けてPhase3の検討に移る
  - 管理者が及び腰になる → 責めない。単独提案への切替か時期をずらす

## 4. Phase 3: エスカレーション(提案拒否・放置の場合)

**内部**: 上部組織(理事会・本部)や監査/ハラスメント窓口があればそちらへ。伝え方は「訴える」ではなく「情報管理・請求管理の懸念、改善のご相談」。直属→法人内上位者・監査窓口→外部、の順序を踏む(いきなり外部は公益通報の保護要件でも不利)。

**外部**:
- 指定権者: 介護保険側の疑いは市区町村・都道府県の介護保険担当課、医療保険側の疑いは地方厚生局
- 個人情報保護委員会: 個人情報の取り扱いに関する相談・情報提供
- 労働基準監督署: 労働条件通知書不交付、パワハラの相談

**公益通報者保護法の使い方**(条文の詳細は `_research/A9_不正類型と労基_公益通報.md` 参照): 現行法は3段階。①内部通報=「通報対象事実が生じている/まさに生じようとしている」で足りる(ハードル最も低い)。②行政機関=「信ずるに足りる相当の理由」が必要。③報道機関等の外部=内部・行政通報で不利益を受けると信じる相当の理由、証拠隠滅のおそれ、通報しないよう不当に要求された、書面通報から20日経過しても通知なし、生命・身体への急迫した危険、のいずれかが必要で最も厳格。要件を満たせば解雇・降格・減給等の不利益取扱いは禁止される。
個人情報保護法・介護保険法・健康保険法はいずれも通報対象法律に含まれることを確認済み(個情法は法律本体別表、介護保険法・健保法は政令〈別表第八号政令〉に掲載)。「個人LINEでの患者情報の扱い」も「保険請求の不正」も通報対象事実になり得る。ただし対象は各法律の刑罰・過料の対象事実に限られ、単なるマナー違反は対象外。
**重要**: 2025年改正法(刑事罰導入等)は**2026年12月1日施行でまだ未施行**。現時点では現行法(2022年施行版)の3段階要件を前提に慎重に進める。

**通報前チェックリスト**:
- [ ] 自分のログ(イニシャル化)は整理できているか
- [ ] 自分自身の個人LINE利用など、自分の関与を正直に整理できているか
- [ ] 弁護士・日本看護協会の相談窓口など事前に確認したか
- [ ] 内部通報の段階を飛ばしていないか

## 5. 各Phase共通: やってはいけないこと

単独で事務局長と対決すること/証拠なしに「不正だ」と断言すること/患者情報を含む書類・画像を私物端末やクラウドに持ち出すこと(自分が個情法上の当事者になり得る)/同僚への言いふらし/SNSへの投稿(日本看護協会も明確に注意喚起、自分自身の懲戒・法的リスクに直結)。

---

## 6. 撤退ライン

**報復の兆候**(シフト外し、担当患者の急な削減、威圧の激化)、**心身への影響**(不眠、強い抑うつ感)――このいずれかが出たら、計画の実行より転職準備を優先する。訪問看護経験者は今も業界全体で売り手市場であり、この職場に固執する理由はない。

ただし辞める場合も、「不正の可能性を知りながら黙って辞めること」で自分がどこまで関与していたことになるのか(自分のLINE利用・自分の記録の実態)は、転職前に自分のためだけに整理しておく。何もしないまま去る選択も一つだが、「知っていて何もしなかった」事実は後で自分に返ってくることがある。
