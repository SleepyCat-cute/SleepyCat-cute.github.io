## 1. 30秒でわかる全体像

(2026年7月、本人へのヒアリングで前提事実を確認・更新)

**起きていること(2つの法的リスク+働き方の問題)**

1. **利用者情報の管理体制の不備**: 個人LINEで氏名(苗字・伏字なし)・電話番号・患部写真・自己負担上限額管理表の写真がやりとりされている(病名は直接書かれていないが、患部写真等から推測は可能)。医療介護専用SNS「MCS」は実はすでに導入されているが、正社員1名(管理者)の1アカウントを全員で共用し、社用メールがないため訪問看護用に作った個人のフリーメール(yahoo.com)で登録・共用しており、個別の利用者管理として機能していない
2. **保険種別(医療/介護)・訪問時間区分の管理不備**により、気づかないうちに基準外の請求が発生しているおそれ(訪問時間の過不足が実際に発生しており、記録上の時間区分が実態に合わせて修正されているか不明な事例あり)

これに加えて、**業務運営・労働条件の問題が5つ**(訪問スケジュールが手書きで一元管理されていない/土日・時間外の全員宛LINE連絡/備品の自己負担とナース服の自宅洗濯/始業8:30と同時刻の訪問開始=前残業/カルテが契約順で探しにくい)。いずれも0円〜小額で対応でき、「管理者の負担軽減」の顔でセキュリティ提案と同じ器に載せられる(詳細は06)。

**放置した場合の帰結(✅確認済みの法令・判例より)**

- 情報漏えい: 病歴等の要配慮個人情報は**1件でも国への報告・本人通知が義務** → 指導・命令 → 命令違反で**法人に最大1億円の罰金**。賠償は病歴を含む事案で**1人30万〜110万円級**。看護師個人にも刑事罰(保助看法、退職後も継続)
- 不正請求認定: **返還+40%加算金** → **指定取消(5年間再指定不可)**=事業継続不能。「知らなかった」でも**重過失なら故意と同視**して不正認定されうる

**対策は0円から始められる**

- 医療介護専用SNS「MCS」は**すでに無料で使える状態にある**。新規導入ではなく、個別アカウント化+運用ルール+誓約書に**整え直すだけ** → 今月から費用ゼロで是正可能(連携クリニックからも全員分のアカウント作成を依頼されており、後押し材料になる)
- 請求側は「**自主点検 → 過誤調整(自主返還)**」なら加算金・処分・公表なしの正規ルート。監査で発覚した場合との差は歴然

**進め方**

Phase 0(自衛)→ Phase 1(味方づくり)→ Phase 2(0円案の正式提案)→ 合意なら導入/拒否・放置なら Phase 3(段階的エスカレーション)。全期間を通じて撤退ライン(報復の兆候・心身への影響)を監視する。

## 2. リスク早見表(何を破ると、何が起きるか)

| 問題 | 主な根拠法令 | 起きうる結果(✅一次ソース確認済み) |
|---|---|---|
| 個人LINEで氏名(苗字)・電話番号・患部写真・自己負担上限額管理表をやりとり/MCSアカウントを複数人で共用 | 個人情報保護法23・24・26条/医療情報システム安全管理ガイドライン第7.0版(「管理されていない端末でのBYODは許可してはならない」「識別・認証情報は本人しか知り得ない状態に保つこと」第14章)/保助看法42条の2/運営基準 | 1件でも個情委報告・本人通知義務 → 指導・命令 → 命令違反で個人「1年以下拘禁刑/100万円以下罰金」・**法人「1億円以下の罰金」**(178・184条)。民事賠償: 病歴含む判例で1人30万円(乳がん臨床写真)、実害ありで110万円。看護師個人は6月以下拘禁刑/10万円以下罰金(親告罪・退職後も継続)。事故対応実費は調査だけで300〜400万円規模(JNSA調査) |
| 時間区分・保険種別が未管理のままの請求 | 介護保険法22条3項・77条1項6号・70条2項6号/健康保険法58条3項・95条3〜5号/厚労省監査マニュアル | 監査で不正認定 → **返還+40%加算金** → 指定取消(**5年欠格**、取消なら遡及全額返還も)。「実際に行った時間に対応する単位数を超えた請求」は不正の典型例と厚労省が明示。**重過失は故意と同視**。実例: 東京都内ステーションが約1,056万円の不正請求で全指定取消(2024年12月)。悪質なら詐欺罪(被害額88万円程度でも経営者逮捕例あり) |

**個人も無関係ではない**: 記録の作成者・署名者は真正性を説明する立場になり、虚偽記録への関与とみなされれば懲戒に加え詐欺罪の共同正犯・幇助、保助看法9条2号による行政処分(戒告・業務停止・免許取消)の対象になり得る。監査での従業者の虚偽答弁はそれ自体が指定取消事由(健保法95条5号)。

**よくある疑問への回答**

- **Q. 指導・命令の段階で是正すれば、178・184条の罰則(拘禁刑・罰金)は発生しないのか?**
  178条が処罰しているのは「命令(委員会が出す是正命令)・緊急命令への違反」であり、個人情報の取扱いそのものへの罰則ではない。流れは①報告・通知→②指導→③勧告→④命令→**⑤命令に違反して初めて刑事罰**、の順(詳細は01)。したがって指導・勧告の段階で是正すれば命令が出される前に収まる可能性が高く、命令が出た後でもそれに従って是正すれば「命令違反」にはあたらないため、178・184条の刑事罰の対象にはならないと解される。
  ⚠️ ただし: (a) 報告・本人通知義務(26条)自体は是正の有無にかかわらず発生する独立の義務で免除されない、(b) 民事賠償・信用毀損のリスクは刑事罰の有無と関係なく残る、(c) どの段階まで進み何をもって「是正十分」とするかは個々の事案でPPCが判断する。断定はできない点に注意(詳細は01)。
- **Q. 監査は何年に1回か、どんな準備が必要か?**
  監査(不正の疑いがある場合の特別な調査)は健康診断のような固定周期のものではなく、実地指導(運営指導)の中で疑義が生じた場合や、通報・苦情をきっかけに行われる(詳細は02)。⚠️実地指導自体の頻度目安は一次資料で確認できていない。「準備」については、実地指導・監査で見られるのは計画書・記録書・提供票・レセプトの整合性であり、直前に取り繕えるものではない。実質的な唯一の備えは、日々の記録を実態と一致させ続けることである(詳細は02)。

## 3. 対策: 3段階プラン(7名規模・月額)

| プラン | 内容 | 月額 | 防げること |
|---|---|---|---|
| **プラン0** | **既存MCSの適正化**(1人1アカウント化・共用フリーメールの廃止)+運用ルール+秘密保持誓約書(無料ひな形) | **0円** | 個人LINE特有のリスク(誤送信・退職者の端末に履歴残存・管理外クラウド保管)に加え、共用アカウントによる識別・認証要件違反(ガイドライン第14章)も解消 |
| **プラン1** | LINE WORKS スタンダード(個人LINEに近い操作感) | 約3,150円 | +操作ログ監査、端末紛失時のアカウント遠隔停止、業務/私用アカウントの分離 |
| **プラン2** | 訪問看護専用電子カルテ(カイポケ等)+業務端末7台+MDM | 約3.3万〜3.4万円 | 写真・記録が端末に残らない設計で漏えいを構造的に排除。**請求管理・スケジュール・オンコールも一元化**(管理者の負担軽減と請求ミス防止を同時に実現) |

入口は必ず**プラン0(0円)**。「新しい物を導入してほしい」ではなく「すでにあるMCSをクリニックにも頼まれている形(個別アカウント)に整えるだけ」という、より通りやすい提案として持ち込む。

## 4. 行動計画: Phase 0〜3 + 撤退ライン

```
Phase0 自衛と現状把握(〜2週間) → Phase1 味方づくり(2〜4週間) → Phase2 正式提案(1〜2ヶ月)
                                                                        │
                                                    ┌───────合意 / 拒否・放置───────┐
                                                    ▼                              ▼
                                              導入・定着                    Phase3 エスカレーション
                                                                          (内部→行政→外部の順)
【全Phase共通】報復の兆候 or 心身への影響 → 撤退ラインで転職準備に切替
```

- **Phase 0(今日から)**: 自分は個人LINEで患者情報を送らない(受信履歴は慌てて消さない)。事実ログ開始(イニシャル化)。自分の担当訪問の実滞在時間と記録の突合。労働条件通知書等4種の書類開示を求める。録音準備。
- **Phase 1**: 疲弊した管理者を「オンコール・訪問割当・時間管理が楽になる話」として最初の味方に。パートの温度感を確認。「不正をなくそう」ではなく「時間管理を分かりやすくしよう」という言葉選び。
- **Phase 2**: 説得資料(05)を使い、可能なら管理者と連名で0円案を提案。面談は録音し、当日中にメールで議事メモ化(最大の証拠化手段)。
- **Phase 3(拒否・放置時)**: 内部(上部組織・窓口)→ 指定権者(介護=自治体、医療=地方厚生局)・個人情報保護委員会・労基署 → 公益通報者保護法の3段階要件に沿って慎重に。個情法・介護保険法・健保法はいずれも通報対象法律(✅確認済み)。
- **撤退ライン(常時)**: シフト外し・威圧の激化・不眠や強い抑うつ → 計画より転職準備を優先。訪問看護経験者は売り手市場であり、この職場に固執する理由はない。

- 本資料は法律相談の代替ではない。エスカレーションや退職・通報など重要な判断の前には弁護士(法テラス等)への相談を推奨。
- **この資料に職場名・実名を書き足さない**。患者情報の現物(写真・氏名入り記録)をこのフォルダにも私物端末にも保存しない。
