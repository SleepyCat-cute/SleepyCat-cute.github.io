# 保険種別と訪問時間の要件・不正請求リスク

作成日: 2026-07-05

---

## 1. この文書の使い方

訪問看護の時間区分・報酬ルールを理解し、自分が不正請求に巻き込まれていないか安全に確認するための資料です。第2章で正しいルール、第3章で線引き、第4章で確認手順、第5〜6章で気づいた場合の対応をまとめています。

---

## 2. 早見表:保険種別ごとの時間区分と報酬

**最重要**:利用者ごとに適用保険が「医療」か「介護」かで、時間区分・単位/点数・算定ルールが全く別物です。取り違えたまま管理すると、気づかないうちにルール違反の請求が発生します。

### 2-1. 介護保険:訪問看護費の時間区分と単位数(指定訪問看護ステーションの場合)

| 時間区分 | 単位数 |
|---|---|
| 20分未満 | 313単位 |
| 30分未満 | 470単位 |
| 30分以上1時間未満 | 821単位 |
| 1時間以上1時間30分未満 | 1,125単位 |

(准看護師の場合は上記の90/100。理学療法士等が行う場合は別区分・293単位/回)

✅出典:厚労省「介護報酬の算定構造」(社保審-介護給付費分科会 第239回 R6.1.22 参考資料2-1)。令和8年6月改定は処遇改善加算新設が中心で基本単位数の見直しは確認できていません。

### 2-2. 医療保険:訪問看護基本療養費(Ⅰ)イ(保健師・助産師・看護師による場合)

| 区分 | 金額 |
|---|---|
| 週3日目まで | 5,550円 |
| 週4日目以降 | 6,550円 |

(1割負担555円/1,110円、2割負担1,110円/1,310円、3割負担1,665円/1,965円。週3日目まで/週4日目以降の順)

✅出典:厚労省告示「訪問看護療養費に係る指定訪問看護の費用の額の算定方法」(令和8年度改定・改正前後対照表)。令和6年6月改定・令和8年度改定を通じて金額は不変。

**訪問時間の基準(告示注3・原文)**:

> この場合の適切な時間の指定訪問看護とは、**30分以上を標準とし、20分を下回らないものであること**。

告示原文の下限規定です。「30分〜90分程度」の目安のうち90分側は告示に上限記載がなく運用実態に基づくもので、確定しているのは「20分を下回ってはならない」という下限です。

**週3日制限とその例外**:

医療保険の訪問看護は原則週3日までですが、以下のいずれかに該当する利用者は週4日以上の訪問が可能です。

| 例外区分 | 内容 |
|---|---|
| 別表第7 | 厚生労働大臣が定める疾病等(いわゆる難病等・末期の悪性腫瘍等) |
| 別表第8 | 別に厚生労働大臣が定める状態等(気管カニューレ使用・真皮を超える褥瘡等、特別管理加算の対象状態) |
| 特別訪問看護指示書 | 急性増悪等により主治医が一時的に頻回の訪問看護が必要と認めた場合。原則月1回・14日を限度に交付 |

✅出典:同告示(注1・注6)。別表第7・第8の詳細リスト原文は未確認のため、個別判断は指示書・計画書と管理者・主治医への確認によってください。

要介護認定・特定疾病該当・末期がん等の状態変化で適用保険は医療⇄介護に切り替わります。見落として旧ルールのまま管理すると基準外の請求が発生するため、「今どちらの保険か」を訪問のたびに確認してください。

## 3. どこからが不正か:白・グレー・黒の3段階

厚労省「介護保険施設等に対する監査マニュアル」(令和6年4月)は、不正請求を次のように定義しています。

> 不正請求とは、法令や基準に違反し、かつそれを偽って報酬を請求することです。(中略)サービスの所要時間によって単位数が定められている場合に**実際にサービスを行った時間に対応する単位数を超えた単位数により請求した場合等**がこれに当たります。

✅出典:厚労省老健局「介護保険施設等に対する監査マニュアル」(令和6年4月、老発0405第3号通知添付)

この定義と、介護保険の所要時間の算定原則を踏まえると、実務上は次の3段階で整理できます。算定原則の原文は次のとおりです(✅原文確認済み)。

> 指定訪問看護事業所の看護師等が指定訪問看護を行った場合に、**現に要した時間ではなく、訪問看護計画書に位置付けられた内容の指定訪問看護を行うのに要する標準的な時間で所定単位数を算定する**。

✅出典:老企第36号(訪問看護費の項)に基づく算定原則。東京都福祉局「指定訪問看護事業 指導検査基準」(令和7年4月1日適用)に同文の点検項目として掲載されており、行政の指導検査もこの観点で行われます。兵庫県「訪問看護・介護予防訪問看護の手引き」(令和6年6月)にも同文。

| 区分 | 状態 | 具体例 |
|---|---|---|
| **白(適法)** | 看護提供が計画内容に見合い、記録も実態通り | 計画30分の訪問で看護提供の実質(観察・処置・記録等)は満たされ、実滞在がたまたま数分短い程度 |
| **グレー(是正すべき管理不備)** | 不正と断定はできないが放置すれば黒化しうる | 計画時間と実態の乖離が常態化しているのに計画を見直さず同じ区分で請求継続。保険種別・時間要件が管理されておらず誰も気づかない |
| **黒(不正請求)** | 監査マニュアル定義に該当 | (a)記録・実績に虚偽の時間を記載、(b)乖離常態化を放置し計画を見直さず上位区分で請求継続、(c)未提供サービスの請求 |

**具体的なケースの当てはめ:利用者都合による早期終了**

「利用者の外出準備等の都合で、了承のうえ予定より早く訪問を終了した」結果、計画上の時間区分に対して実滞在時間が大きく下回る(例:30分区分の計画に対し実滞在18分)という記録が見つかることがあります。考え方の整理:

- 看護提供の実質(観察・処置等、計画上必要な内容)が実際に完了していたのであれば、「現に要した時間ではなく標準的な時間で算定する」原則(前掲)により、直ちに不正にはなりません(白に近い)。
- 必要な看護が完了しないまま利用者都合で終了したにもかかわらず、記録・請求区分を見直さずに計画通りの区分で請求を続けているなら、乖離の常態化(グレー、放置すれば黒化)にあたり得ます。
- **この記録が実態に見合う時間区分に修正されているかどうかを確認・判断すべき役割(事務なのか、訪問した看護師本人なのか)が定められていないこと自体が管理不備**です。役割分担を明確にする提案は「04_行動計画」を参照してください。

⚠️ この種の個別ケースの適法性そのものへの一次資料上の直接の当てはめ例は確認できていません。上記は前掲の算定原則・不正請求の定義からの論理的な整理であり、断定はできない点に留意してください。

**重要な補足**:介護保険の訪問看護費は「実測時間」ではなく「計画上の標準的な時間」を基準に算定するのが原則です(前掲・原文確認済み)。したがって実滞在時間が計画よりわずかに短い程度は直ちに不正になりません。不正になるのは上表(a)〜(c)のような場合です。

**「知らなかった」では済まない場合がある点にも注意してください。**

> 実務では、不正請求が疑われる場合について、故意が認められた場合に限り不正請求と認定し、過失にすぎない場合は過誤調整として取り扱うという運用がみられるようです。しかしながら、過失である場合においても一切不正が認定できないわけではないことには注意してください。とりわけ**重過失が認められる場合においては、故意とほぼ同視し、不正と認定することを妨げられるべきではありません**。

✅出典:同監査マニュアル。時間要件の管理体制がないまま基準を満たさない請求が常態化していれば、重過失として不正認定されるリスクがあります。

## 4. あなたが安全に確かめる手順

事業所全体を調査する権限も義務もあなたにはありません。自分自身の業務範囲内で、正当な業務行為として事実確認を行う手順です。

1. 自分の直近の訪問について、実滞在時間の記憶・メモと、記録書・提供票の記載時間を突き合わせる。**対象は自分が担当した訪問に限定。**
2. 確認は自分の業務記録(自分が作成・関与した記録書、スケジュール、メモ)の範囲内で行う。他の職員が担当した利用者のレセプト・記録の閲覧は不要。
3. 他人のレセプト・記録を漁らない。書類の原本・コピーを事業所外に持ち出さない。目的外の閲覧・持ち出しは新たな守秘義務違反になる。
4. 私的メモに患者の実名は書かず、イニシャル化する。
5. 違和感があれば管理者に「この時間区分で合っていますか」「今どちらの保険ですか」と質問し、日時と回答を自分のメモに残す。**この質問自体は正当な業務行為。**

ポイントは「自分の業務範囲内の事実確認」と「他人の記録の詮索」の区別です。前者は行ってよく、後者は避けるべきです。

## 5. 虚偽記録に自分が関与してしまっていた場合

管理者や先輩の指示、事業所の慣行に従って、実態と異なる時間を記載した記録に自分が署名・作成者として関わっていた可能性に気づくことがあるかもしれません。

**知らずに署名していた場合のリスク**:作成者・署名者として名前が残る以上、後に不正請求が発覚した際にはその記録の真正性を説明する立場になり得ます。虚偽記録への関与とみなされれば、懲戒処分に加え詐欺罪の共同正犯・幇助、保健師助産師看護師法9条2号(業務に関し犯罪又は不正の行為があった者)に基づく行政処分(戒告・業務停止・免許取消)の対象になり得ます。

**気づいた後に取るべき行動**:

- 以後の記録は実態通りに書く(最優先)。
- 気づいた事実を管理者に伝え、過去の記録の修正・是正方法(自主点検・自主返還等)を申し出る。
- 申し出た日時・相手・内容・経過を自分のメモに記録しておく(後日、自分の対応の正当性を示す材料になる)。

**放置がいちばん危険な理由**:気づいていながら何もしない状態は「知りながら是正しなかった」という評価につながります。重過失は故意とほぼ同視され不正認定され得るため、早く行動することが自分自身を守ります。

## 6. 事業所として発覚した場合の帰結の差

同じ「時間要件を満たさない請求」でも、発覚の経緯によって事業所が受ける結果は大きく異なります。

| 経緯 | 結果 |
|---|---|
| **自主点検・自主返還(過誤調整)** | 自ら誤りに気づき返還すれば、単純な過誤調整として扱われる余地がある |
| **監査で不正と認定** | 返還額に加え**40%の加算金**(介護保険法22条3項)。指定取消事由(同法77条1項6号)、取消後**5年間再指定不可**(同法70条2項6号)。医療保険側も同様に40%加算金(健保法58条3項)・指定取消事由(同法95条3号)が明文 |

✅出典:介護保険法22条3項・70条2項6号・77条1項6号、健康保険法58条3項・95条3号(e-Gov法令API原文確認)、厚労省「介護保険施設等に対する監査マニュアル」

**実例(一次資料で確認できている返還額)**:

東京都内の訪問看護ステーション「べんてん」(株式会社たんぽぽ運営)は令和6年12月27日付で指定取消処分を受けました。虚偽のサービス提供記録の作成、実績のない利用者への虚偽請求、准看護師を看護師と偽った請求などにより、不正請求額は約1,056万円(介護保険法・生活保護法分約552万円、生活保護法医療扶助分約504万円)。指定訪問看護事業所・指定介護予防訪問看護事業所・指定医療機関の指定がすべて取り消されました。小規模な事業所でも記録の不正の積み重ねで事業継続が不可能になる実例です。✅出典:東京都福祉局プレスリリース(2024年12月)。

なお、大手事業者による過大請求疑惑(2024〜2026年報道)の試算額(数千万〜数十億円)は**会社側の特別調査委員会による試算であり行政による確定額ではありません**。行政処分が確定しているのは「べんてん」のような自治体公表事例です。

早期に自ら気づき是正すれば「自主点検・自主返還」で済む可能性がある一方、管理不備を放置し監査で発覚すれば40%加算金・指定取消・事業所名公表、悪質なら刑事責任にまで発展し得ます。差は「気づいた時に動いたか」で分かれます。

## 6-1. 実地指導と監査の違い・頻度・「準備」の実質

**実地指導(運営指導)と監査は別物です。**

| | 実地指導(運営指導) | 監査 |
|---|---|---|
| 性格 | 定期的・計画的な通常のチェック | 不正の疑いがある場合に行われる特別な調査 |
| トリガー | 指定権者の計画に基づく通常巡回 | 通報・苦情・実地指導中に生じた疑義など |
| 結果が軽い場合の対応 | 口頭・文書指導 | ― |
| 結果が重い場合の対応 | (疑義があれば監査に移行) | 改善勧告・命令 → 効力停止 → 指定取消 |

**頻度**:⚠️実地指導の実施サイクルについて「◯年に1回」等の確定的な数字は、今回の調査では一次資料上確認できていません。監査自体は健康診断のような固定周期のものではなく、実地指導中の疑義発覚や通報・苦情を契機に行われるものです。「次はいつ来るか」を正確に予測する方法はないと考えておくのが安全です。

**準備として実務上できること**:実地指導・監査で確認されるのは、訪問看護計画書・記録書・提供票・レセプトの整合性です(いずれも完結の日から2年間の保存義務があります、01参照)。これらは**直前に取り繕えるものではなく**、日々の記録が実態と一致した状態を保ち続けることが実質的な唯一の備えです。自分の担当訪問について実滞在時間と記録を突き合わせる習慣(4章)自体が、最も有効な「準備」になります。
