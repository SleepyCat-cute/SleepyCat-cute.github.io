## 1. Medical Care Station(MCS/エンブレース株式会社)

### 概要・位置づけ
- **結論**: MCS(メディカルケアステーション)は、エンブレース株式会社が運営する「完全非公開型(招待制)の医療介護専用SNS」。全国90,000人以上の医療介護従事者、200以上の医師会等で採用されている。患者・利用者ごとにトークルームを作り、招待・承認された関係者のみが参加できる仕組み。
- **出典**: https://about.medical-care.net/html/service/ 、https://about.medical-care.net/html/pricing/
- **確認状態**: ✅公式確認

### 無料プラン(フリープラン)の範囲
- **結論**: 完全無料で、ユーザー登録数(利用施設・職員数)に制限なし。ただし以下の制限あり。
  - 患者・利用者登録数:100件まで
  - グループ(トークルーム)登録数:30件まで
  - ストレージ容量:30GB/ユーザー
  - 添付ファイル:1ファイル50MBまで、1投稿3ファイルまで
  - メッセージ・ファイル検索:直近90日以内のみ
  - 未読メッセージ一括確認:月80件まで
  - ビデオ通話:開始不可(参加のみ可)
  - MCS ID作成数:3アカウントまで
- 有料の「ビジネスプラン」は月額300円/ユーザー(税抜)で、ほぼ全機能が利用可能。90日間無料トライアルあり。
- **出典**: https://about.medical-care.net/html/pricing/
- **確認状態**: ✅公式確認(2026年7月時点WebFetch確認)

### 無料である理由(運営費の出所)
- **結論**: 運営費はMCS公式アプリのパートナー企業からの収益や、医療システム受託開発の収益等で賄っており、医療・介護従事者は無料で利用できる仕組み。
- **出典**: https://support.medical-care.net/hc/ja/articles/360007340834
- **確認状態**: ✅公式確認

### セキュリティ・ガイドライン準拠
- **結論**: 通信はTLS1.2で暗号化。運用インフラは全て日本国内で完結しており、国外への業務委託・データ移管は行っていない。厚生労働省「医療情報システムの安全管理に関するガイドライン」、経産省・総務省「医療情報を取り扱う情報システム・サービスの提供事業者における安全管理ガイドライン」に準拠していると明記。第三者認証としてISMS認証(ISO/IEC 27001、認証番号ISMS2700127017)を取得。プライバシーマークの取得は公式ページ上では確認できず。
- **出典**: https://about.medical-care.net/html/service/security.html 、https://about.medical-care.net/html/service/guideline_docs.html
- **確認状態**: ✅公式確認(TLS1.2・国内運用・厚労省ガイドライン準拠・ISMS認証)

### 小規模ステーションでの実用性メモ
- 7名規模であれば患者登録100件・グループ30件の上限に達する可能性はあるが、担当患者数によっては無料枠で運用可能な場合が多い。上限に達した場合のみ月額300円/ユーザーのビジネスプランへの移行を検討すればよく、初期投資ゼロでセキュリティ対策を始められる点が最大の強み。

## 2. LINE WORKS

### 現行価格(2026年7月時点)
- **結論**:
  | プラン | 年額契約(月換算) | 月額契約 | ユーザー数上限 | ストレージ |
  |---|---|---|---|---|
  | フリー | ¥0 | ¥0 | 30人まで | 合計5GB(追加不可) |
  | スタンダード | ¥450/ユーザー/月 | ¥540/ユーザー/月 | 無制限 | 1TB |
  | アドバンスト | ¥800/ユーザー/月 | ¥960/ユーザー/月 | 無制限 | 基本100TB |
  - フリープランは初期費用なし、最少人数条件なしで1人から利用可能。ただし2023年の仕様変更でフリープランは「30人以下」に制限された(旧仕様ではより緩やかな条件だった)。
  - 年額契約は月額契約より約20%割安。
- **出典**: https://line-works.com/pricing/
- **確認状態**: ✅公式確認(2026年7月時点WebFetch確認)

### 7名規模での試算
- フリープラン(30人まで)の範囲内に収まるため、**月額0円でも運用可能**。ただしフリープランは管理者機能(監査ログの一部・詳細な権限設定等)に制限があるため、本格的なログ監査・遠隔管理を求める場合はスタンダードプラン(年額契約:450円×7名=月額3,150円)が現実的な最小コスト。
- **確認状態**: ⚠️フリープランの管理機能制限の詳細範囲(監査ログがどこまで見られるか等)は二次情報のみで、公式のプラン比較表での機能差分の一次確認までは行っていない

### 個人LINEとの違い
- **結論**:
  1. **アカウント管理**: LINE WORKSのIDは電話番号と紐付かず、管理者がID・パスワードを発行。組織のドメイン管理下でメンバーを一括管理できる。
  2. **ログ監査**: 管理者向け「監査」機能があり、メンバーの操作履歴、管理者画面への接続IPアドレス・作業内容等を記録・確認できる。
  3. **遠隔データ消去**: 端末の紛失・盗難時に、管理者が遠隔操作でLINE WORKSアカウントを停止し、アプリ内データへのアクセスを遮断できる。外部MDMと連携すれば特定端末からのみログイン許可する等の制御も可能。
  4. 個人LINEにはこれらの组織管理・監査・強制ログアウト機能が存在しない。
- **出典**: https://help.worksmobile.com/ja/admin-guides/audit/ 、https://line-works.com/feature/admin/ 、https://www.mobileworkplace.jp/column/lineworks-line_difference
- **確認状態**: ✅公式確認(監査機能・管理機能ページ)、⚠️遠隔データ消去の詳細な仕様(アプリのみ消去かデバイス全体かなど)は二次情報での言及

### 医療・介護・訪問看護での導入実績
- **結論**: LINE WORKS公式サイトに医療・福祉・介護業界向けの事例ページがあり、訪問看護ステーションでの具体的な導入事例が複数掲載されている。
  - **株式会社松(香川県高松市、訪問看護ステーション)**: 従来は患者のセンシティブな情報を個人LINEでやり取りしており、「スマホを紛失した際の情報セキュリティ」が以前から懸念されていた。LINE WORKS導入後は、患者ごとのトークグループで写真・動画を使った遠隔相談が可能になり、既読機能で連絡漏れを防止、緊急時の人員確保も電話から一斉トークに切り替えて効率化した。
  - 医療法人社団健育会「ひまわり在宅サポートグループ」(宮城県、訪問看護・訪問介護等)、鈴木内科居宅介護支援事業所(札幌市、訪問看護等10事業)でも導入。
- **出典**: https://line-works.com/cases/matsu-h/ 、https://line-works.com/cases/industry/medical/ 、https://line-works.com/solutions/long-term-care/
- **確認状態**: ✅公式確認(公式事例ページ・WebFetch確認)

### 小規模ステーション向けの実務メモ
- 「個人LINEの延長」から一足飛びに電子カルテ導入は経営層に心理的ハードルが高い場合、LINE WORKSは「使い方は個人LINEとほぼ同じだが、組織管理・監査・遠隔消去ができる」ため導入の説得材料になりやすい。7名規模ならフリープランで開始し、監査ログの必要性が高まった時点でスタンダードプラン(月額3,150円程度)へ移行するのが現実的な導線。

## 3. 訪問看護専用電子カルテ(iBow・カイポケ・ワイズマン等)

### iBow(株式会社eWeLL)
- **結論**: 初期費用0円(サーバー構築費無料)。基本料金は1事業所あたり月額18,000円(ディスク容量・サーバー管理・セキュリティ・カスタマーサポート・法令対応込み)。レセプト利用料が別途月額1,000円。加えて「訪問看護記録書Ⅱ」の請求確定件数に応じた従量課金(公式サイトでは「事業規模に合わせた従量制」とのみ記載、二次情報では確定件数×100円程度との言及あり)。全国47都道府県で54,000人以上の看護師・療法士が利用。
- **出典**: https://ewellibow.jp/price/
- **確認状態**: ✅公式確認(基本料金18,000円/月・レセプト利用料1,000円/月・初期費用0円)、⚠️従量課金の単価(記録書Ⅱ件数×100円)は二次情報のみで公式ページでの単価明記は確認できず

### iBowの主要機能(写真管理・スケジュール・保険請求)
- **結論**:
  - **写真をシステム内で完結**: iBowはクラウド型で、タブレット等の端末に記録データ・撮影写真を保存しない仕組み。撮影した写真はiBow上の専用サーバーに保存され、端末のロックが解除されても端末内に写真は残らない。ID・パスワードでログインしないとアクセスできないため、端末紛失・盗難時の情報漏洩リスクを抑えられる。
  - **スケジュール一元管理**: 訪問スケジュール管理、AIによる訪問計画書・報告書・スケジュールの自動作成機能あり。
  - **医療/介護保険の請求管理**: 介護保険・医療保険どちらの請求にも対応。介護保険は伝送オプション、医療保険はオンライン請求・オンライン資格確認に対応。
- **出典**: https://ewellibow.jp/i-bow/karte/ 、https://ewellibow.jp/i-bow/receipt/ 、二次情報(digi-mado.jp等の比較記事)
- **確認状態**: ✅公式確認(機能自体の存在)、⚠️「端末に写真が残らない」旨の詳細な技術的説明は公式ページより二次比較記事での記述が中心

### カイポケ訪問看護(株式会社エス・エム・エス)
- **結論**: 月額25,000円(税別)の固定料金制で、1拠点(事業所番号)あたりの課金。利用者数・訪問件数・端末台数が増えても追加料金は発生しない。初期費用・サポート費用は0円。2か所目以降のサテライト事業所は1か所ごとに月額10,000円(税別)が追加。レセプト作成・記録・帳票作成・タブレット対応を含む。
- **出典**: https://ads.kaipoke.biz/price/ 、https://houkan.kaipoke.biz/price/
- **確認状態**: ✅公式確認(2026年7月時点WebFetch確認)

### ワイズマン(訪問看護ステーション管理システムSP)
- **結論**: 公式サイトに具体的な金額の記載なし。料金体系は「5年間使用権パック(システム利用料+クライアント基本料、保守料、法改正対応料、バージョンアップ・リニューアル料を含む)」+「初期導入費用(設定料・導入支援費・操作説明料)」の組み合わせで、事業所ごとの個別見積もり(ヒアリング後1週間程度で見積書提出)。複数の比較サイトでも「料金:要問い合わせ」としか記載されておらず、月額相場は特定できなかった。
- **出典**: https://www.wiseman.co.jp/products/welfare/price/
- **確認状態**: ⚠️公式サイト・二次情報とも具体的金額の記載なし。相場感は不明(要問い合わせのみ)

### 7名規模での月額試算
- **iBow**: 基本料金18,000円 + レセプト利用料1,000円 + 従量課金(件数による、⚠️未確認) ≈ **月額19,000円〜(従量分除く)**。7名がフルタイムで稼働し月間延べ訪問件数が多い場合は従量課金分で上乗せされる可能性がある。
- **カイポケ**: 1拠点固定で**月額25,000円(税別)**。人数・訪問件数によらず定額なので、7名規模なら予算計算がしやすい。
- **ワイズマン**: ⚠️金額非公開のため試算不可。個別見積もりが必須。
- 3社とも初期費用は原則0円〜低額(iBow・カイポケは0円明記、ワイズマンのみ初期導入費用が別途発生)。
- **確認状態**: ✅iBow・カイポケの基本料金は公式確認、⚠️iBowの従量課金部分とワイズマンの金額は未確認

## 4. 業務用スマホ+MDMの相場

### MDM(モバイルデバイス管理)の費用相場
- **結論**: MDM単体の相場は、初期費用0円〜50,000円程度(多くは20,000〜50,000円)、月額費用は1台あたり150円〜2,000円程度で、150〜300円前後の価格帯が中心。リモートロック・リモートワイプ(遠隔データ消去)に機能を絞った安価なプランでは1台月額100円からのものもある。台数が多いほどボリュームディスカウントが効く傾向。
  - 具体例(二次情報): CLOMO MDM(初期費用19,800円+月額300円/台)、mobiconnectエントリープラン(月額換算165円/台)、KDDI Smart Mobile Safety Manager(1ID月額330円・税込、初期費用0円)、OPTiM Biz(月額300円〜/台)
- **出典**: https://www.onamae.com/business/article/52170/ 、https://boxil.jp/mag/a8433/ 、https://biz.kddi.com/content/column/smb/mdm-mobile-security/
- **確認状態**: ⚠️相場感は複数の比較サイト(二次情報)からの集計値。個別製品の価格は各社公式ページでの一次確認までは実施していない

### 業務用スマホ本体+回線費用の相場
- **結論**: 法人向け格安SIM(データ通信のみ、1〜3GB程度)であれば月額1,000円前後から導入可能。1回線から契約でき最低利用期間の縛りがないMVNOも複数存在する。大手キャリアの法人向けプラン(ワイモバイル等)は端末とSIMのセット契約にも対応。既存の私物端末を業務利用させず、事業所として端末を新規調達する場合、本体price(端末代)は別途必要(価格は機種依存のため相場化困難)。
- **出典**: 複数の比較記事(office110.jp、rokemoba.co.jp等)
- **確認状態**: ⚠️二次情報のみ。特定キャリア・MVNOの公式料金ページでの一次確認は未実施

### 7名規模での試算(スマホ本体+回線+MDM)
- **結論**: 端末を新規調達せず既存の私物端末をMDM登録のみする場合は「MDM費用のみ」で、7台×月額200〜300円=**月額1,400円〜2,100円程度**。端末も新規に業務専用で調達する場合は、格安SIM回線(1台月額1,000円前後)+MDM(1台月額200〜300円)で7台合計**月額8,400円〜9,100円程度**(端末本体の分割払い代・初期費用は別途)。
- **確認状態**: ⚠️試算は二次情報の相場レンジを基にした概算であり、公式一次価格での積み上げではない

## 5. 規程ひな形の入手先

### 全国訪問看護事業協会
- **結論**: 公式サイトのニュースページで、厚生労働省・個人情報保護委員会が発出する個人情報保護関連のガイドライン・注意喚起(例:「医療・介護関係事業者における個人情報の適切な取扱いのためのガイダンス」の改正案内、「SNSで実行犯を募集する手口による強盗や特殊詐欺事案に関する緊急対策プランを踏まえた個人情報の適正な取扱いについて」等)を随時掲載・案内している。ただし、協会独自の「個人情報保護規程ひな形」そのものを直接公開しているページは今回の調査では確認できず、多くは厚労省等外部サイトへの案内にとどまる。
- **出典**: https://www.zenhokan.or.jp/new/new126/ 、https://www.zenhokan.or.jp/new/new2083/
- **確認状態**: ✅公式確認(ガイドライン案内の実施自体)、⚠️協会独自の規程ひな形そのものの所在は未確認(会員専用ページ内に格納されている可能性はあるが非会員からは確認不可)

### 個人情報保護委員会「医療・介護関係事業者における個人情報の適切な取扱いのためのガイダンス」
- **結論**: 個人情報保護委員会が公表する医療・介護事業者向けの公式ガイダンス。規程整備について「保有個人データの開示手順を定めた規程その他個人情報保護に関する規程を整備し、苦情への対応を行う体制も含めて、院内や事業所内等への掲示やホームページへの掲載を行うなど、患者・利用者等に対して周知徹底を図る」ことを求め、プライバシーポリシー(個人情報保護方針)の策定・公表も求めている。委託先の監督(受託者が個人情報を適切に取り扱っているか定期確認する)についても詳細記載あり。ひな形そのものではないが、規程作成時に参照すべき公式の「型」を示す一次資料。
- **出典**: https://www.ppc.go.jp/personalinfo/legal/iryoukaigo_guidance/
- **確認状態**: ✅公式確認(2026年7月時点WebFetch確認)

### 大分県商工会議所連合会「個人情報保護法各種規程サンプル集」

## 6. 個人版LINEの業務利用に関する位置づけ・注意喚起

### LINEヤフーの利用規約上の位置づけ
- **結論**: LINE利用規約第15条(8)で「個人情報、登録情報、利用履歴情報等の第三者に関する情報を不正に収集、蓄積したり、開示、提供したりする行為」を明確に禁止行為として規定している。また第2.4項では、事業者のために利用する場合は当該事業者も利用規約に同意している必要があるとされ、法人・業務利用自体を一律禁止しているわけではないが、第三者(患者・利用者)の個人情報を本人の同意なくやり取りする行為は、利用規約上の禁止行為に抵触しうる。LINE自体に「医療情報の業務利用を明示的に禁止する」条項は確認できなかったが、個人情報の不正な収集・開示を禁じる一般条項が該当する。
- **出典**: https://terms.line.me/line_terms?lang=ja
- **確認状態**: ✅公式確認(規約条文自体)、⚠️「患者の患部写真や病名のやり取り」がこの条項に該当するかの法的評価は二次的な解釈であり公式見解ではない

### 厚生労働省ガイドラインにおけるBYOD・SNSの扱い
- **結論**: 厚労省「医療情報システムの安全管理に関するガイドライン」は、2017年5月公表の第5版で大きな方針転換があり、BYOD(私物端末の業務利用)を原則禁止とした。最新版は第7.0版(2026年6月改定、令和8年6月)。SNS等を使った医療介護連携を行う場合の注意点として、二次情報の複数記事が共通して「(1)クローズドSNS(非公開型・招待制のSNS)を利用する、(2)通信経路・端末認証にTLS1.2等の暗号化技術を使用する、(3)私物端末の利用(BYOD)を避ける」の3点を挙げている。MCS(メディカルケアステーション)はこの「クローズドSNS」の代表例として複数の自治体・医師会文書で紹介されている。
- **出典**: https://www.mhlw.go.jp/stf/shingi/0000516275_00006.html(第7.0版の存在・改定時期)、二次情報(xtech.nikkei.com等、BYOD原則禁止化の経緯)
- **確認状態**: ✅公式確認(第7.0版の存在・改定時期)、⚠️「クローズドSNS利用・TLS1.2・BYOD回避」の3点はガイドライン本文PDFでの一次条文確認はできておらず、複数の二次解説記事の一致から集約した内容

## 8. まとめ表:7名規模の月額3段階試算

7名規模(常勤換算)の訪問看護ステーションを想定し、防げるリスクと合わせて3段階で試算する。金額は2026年7月時点の公式確認情報を基本とし、⚠️は未確認・概算部分。

### プラン0「0円対策」: MCS無料版+運用ルール+誓約書
| 項目 | 内容 | 月額 | 確認状態 |
|---|---|---|---|
| コミュニケーション基盤 | MCS(メディカルケアステーション)フリープラン | 0円 | ✅ |
| 規程・誓約書 | 個人情報保護規程・秘密保持誓約書(公開ひな形を叩き台に自作) | 0円 | ✅(ひな形の存在) |
| 運用ルール | 「患者情報はMCSでのみやり取りし、個人LINEは業務利用禁止」等の社内ルール策定 | 0円 | - |
| **合計** | | **0円** | |
- **防げるリスク**: MCS利用により、患部写真・氏名(苗字)・病名等の患者情報を「非公開型・招待制」の環境でやり取りできるようになり、個人LINEでの漏洩リスク(誤送信、退職者のトーク履歴残存、LINE社の管理外での情報保管等)を大幅に低減。誓約書により職員の法的な自覚・抑止力を高める。【追記2026-07-05】本人へのヒアリングで、対象ステーションはMCSを実際にはすでに導入済みだが、1アカウントを全員で共用し個人のフリーメールで登録している誤運用が判明。この場合の防げるリスクは「導入」ではなく「個別アカウント化」によって得られる(詳細は`_research/A5_セキュリティ関連法規.md`5章)。
- **限界**: 組織的なログ監査・遠隔デバイス管理機能は持たない(MCSは非公開型トークルームでの情報共有が中心)。端末紛失時の遠隔データ消去機能はMCS自体にはなく、別途端末側のセキュリティ(画面ロック等)に依存する。

### プラン1「小規模標準」: LINE WORKS等、月数千円
| 項目 | 内容 | 月額(7名) | 確認状態 |
|---|---|---|---|
| コミュニケーション基盤 | LINE WORKSスタンダードプラン(年額契約) | 450円×7名=3,150円 | ✅ |
| 規程・誓約書 | プラン0と同様 | 0円 | ✅ |
| **合計** | | **約3,150円** | |
(フリープランのままなら0円運用も可能。ただし監査機能等が制限される)
- **防げるリスク**: プラン0の効果に加え、管理者による操作履歴監査(誰が・いつ・どこから接続したか)、端末紛失・盗難時のアカウント遠隔停止が可能になる。個人LINEからの「完全な移行」の受け皿として機能し、業務用アカウントと私用アカウントを制度的に分離できる。
- **限界**: 電子カルテのような医療/介護保険請求管理機能は持たない。写真データはLINE WORKS上のストレージに残るため、電子カルテほどの「端末に一切残さない」設計にはなっていない。

### プラン2「フル装備」: 電子カルテ+端末+MDM、月数万円
| 項目 | 内容 | 月額(7名) | 確認状態 |
|---|---|---|---|
| 訪問看護専用電子カルテ | カイポケ訪問看護(1拠点定額) | 25,000円(税別) | ✅ |
| 業務用端末(格安SIM回線) | 7台(1台あたり月額1,000円前後) | 約7,000円 | ⚠️(二次情報の相場) |
| MDM | 7台(1台あたり月額200〜300円) | 約1,400〜2,100円 | ⚠️(二次情報の相場) |
| **合計** | | **約33,400〜34,100円** | |
(iBowを選ぶ場合は基本料金18,000円+レセプト利用料1,000円+従量課金⚠️のため、端末・MDM込みで概算月額26,400円〜+従量課金分)
- **防げるリスク**: 写真・記録がシステム内(クラウドサーバー)で完結し端末に一切残らないため、端末紛失・盗難時の情報漏洩リスクを構造的に排除。医療保険・介護保険の請求管理も一元化され、業務効率化も同時に実現。MDMにより会社が端末を一元管理し、退職時のアカウント削除・データ消去も確実に実行できる。
- **限界**: 導入・運用コストが最も高く、経営層の説得には「業務効率化・請求ミス防止による副次的なコスト削減効果」も併せて訴求する必要がある。

### 試算の総括
- 「0円でも始められる」(MCS)から「月数千円で個人LINEを代替する組織的管理ができる」(LINE WORKS)、「月3万円強で情報漏洩リスクを構造的に排除しつつ業務効率化する」(電子カルテ+端末+MDM)まで、段階的な投資に応じてリスク低減度が上がる設計。経営層への説明では、プラン0・1は「ほぼコストをかけずに今すぐ始められる対策」、プラン2は「投資対効果(請求効率化・記録の正確性向上)を含めた中長期の検討事項」と整理すると理解を得やすい。
