# 訪問看護ステーション 情報管理・請求管理 改善キット

作成日: 2026-07-05

小規模訪問看護ステーション(常勤2〜3名+非常勤4名規模)で起きている
(1) **個人LINEでの利用者情報(氏名・電話番号・患部写真・自己負担上限額管理表)のやりとり、および医療介護専用SNS「MCS」の誤運用(1アカウントを全員で共用・個人メールで登録)**
(2) **保険種別・訪問時間の管理不備による不正請求の疑い**
に対して、内部にいる看護師が安全に改善を進めるための資料一式です。

## 信頼性の表記ルール

- **✅** = e-Gov法令検索・厚労省・個人情報保護委員会・自治体公表資料など一次ソースで原文確認済み(2026年7月5日時点)
- **⚠️ / 【未確認】** = 二次情報のみ、または試算・類推。断定に使わないこと

説得資料(05)の本文に使っている金額・条文は、すべて✅のものだけで構成しています。

## 取り扱い上の注意

- 本資料は一般的な法令情報の整理であり、法律相談の代替ではありません。エスカレーション(Phase 3)や退職・通報など重要な判断の前には、弁護士(初回無料の労働・医療分野相談窓口や法テラス等)への相談を推奨します。
- 資料内に特定の施設・個人を識別する情報は含めていません。**この資料に職場名や実名を書き足さないでください**(そのまま第三者に見せられる状態を保つため)。
- 患者情報の現物(写真・氏名入り記録など)をこのフォルダにも私物端末にも保存しないでください。
